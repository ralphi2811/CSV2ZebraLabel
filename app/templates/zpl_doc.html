Guide de base du langage ZPL (Zebra Programming Language)

Introduction

Le ZPL (Zebra Programming Language) est un langage de programmation créé par Zebra Technologies pour contrôler leurs imprimantes d'étiquettes. Chaque commande ZPL commence par le caractère ^ (circonflexe).

Structure de base

^XA  // Début de l'étiquette
^...  // Commandes ZPL
^XZ  // Fin de l'étiquette

Commandes essentielles

Position et orientation

Commande Description Exemple
^FO Position de départ (Field Origin) ^FO50,50 (x=50, y=50)
^FW Orientation de l'étiquette ^FWN (Normal), ^FWR (Rotation 90°)

Texte et polices

Commande Description Exemple
^FD Données du champ (Field Data) ^FDHello World
^CF Police et taille ^CF0,30 (Police 0, hauteur 30)
^FB Bloc de texte avec retour à la ligne ^FB300,3 (Largeur 300, max 3 lignes)

Codes-barres

Commande Description Exemple
^BY Paramètres du code-barres ^BY3 (Largeur module = 3)
^BC Code 128 ^BCN,100,Y,N
^BQ QR Code ^BQN,2,4

Exemple complet

^XA
^FO50,50^CF0,30^FDNom: $1^FS
^FO50,100^CF0,30^FDPrénom: $2^FS
^FO50,150^BY3
^BCN,100,Y,N^FD$3^FS
^XZ

Cet exemple crée une étiquette avec :

  • Un champ "Nom" à la position (50,50)
  • Un champ "Prénom" à la position (50,100)
  • Un code-barres Code 128 à la position (50,150)

Variables

Dans CSV2ZebraLabel, vous pouvez utiliser des variables ($1, $2, etc.) qui seront remplacées par les valeurs des colonnes de votre fichier CSV/Excel :

  • $1 = première colonne
  • $2 = deuxième colonne
  • etc.

Pour plus d'informations sur le langage ZPL, consultez la documentation officielle Zebra.
